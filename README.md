## GPT

### Setting Up ShellGPT
 - apt-get install git jq
 - export OPENAI_API_KEY= <Key from OpenAI>
 - export CHATGPT_KEY= <Key from Open AI>
 - pip install shell-gpt
 - apt install python3-venv
 - mkdir shellgpt
 - cd shellgpt
 - python3 -m venv shellgpt
